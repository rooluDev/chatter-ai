## 1. 디자인 토큰

### 1.1 색상

```css
/* 배경 */
--bg-primary:    #FFFFFF;   /* 메시지 영역, 카드 */
--bg-secondary:  #F4F5F7;   /* 페이지 배경 */
--bg-sidebar:    #1A1D21;   /* 사이드바 (다크) */
--bg-sidebar-hover: #27292D;
--bg-sidebar-active: #1164A3;

/* 텍스트 */
--text-primary:  #1D1C1D;
--text-secondary:#616061;
--text-on-dark:  #D1D2D3;   /* 사이드바 내 텍스트 */
--text-on-dark-active: #FFFFFF;

/* 강조 */
--color-primary: #1164A3;   /* 버튼, 링크 */
--color-primary-hover: #0D4F85;
--color-danger:  #E01E5A;   /* 삭제, 경고 */
--color-success: #2BAC76;   /* 온라인 상태 */
--color-warning: #ECB22E;   /* 자리비움 */
--color-ai:      #F0F4FF;   /* AI 메시지 배경 */
--color-ai-border: #C7D8F8; /* AI 메시지 테두리 */

/* 보더 */
--border-light:  #E8E8E8;
--border-medium: #C4C4C4;
```

### 1.2 타이포그래피

```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;

--text-xs:   11px;   /* 타임스탬프, 뱃지 숫자 */
--text-sm:   13px;   /* 보조 텍스트, 레이블 */
--text-base: 15px;   /* 메시지 본문 */
--text-lg:   18px;   /* 채널명, 섹션 헤더 */
--text-xl:   22px;   /* 페이지 제목 */

font-weight-normal: 400;
font-weight-medium: 500;
font-weight-bold:   700;
```

### 1.3 간격 / 크기

```css
--sidebar-width:     240px;
--gnb-height:        64px;
--channel-header-height: 56px;
--input-area-height: 72px;   /* 기본. 멀티라인 시 자동 확장 */
--border-radius-sm:  4px;
--border-radius-md:  8px;
--border-radius-lg:  12px;
--border-radius-full: 9999px;  /* 아바타, 뱃지 */
```

## 2. 공통 레이아웃

### 2.1 전체 레이아웃 구조

`/channels` 이후 모든 페이지에 적용되는 고정 레이아웃.

```
┌───────────────────────────────────────────────────────────────┐
│                         GNB (64px)                            │
├─────────────────────┬─────────────────────────────────────────┤
│                     │         채널 헤더 (56px)                 │
│    사이드바          ├─────────────────────────────────────────┤
│    (240px)          │                                         │
│    ·채널 목록        │          메시지 영역                     │
│    ·DM 목록         │       (flex-grow: 1, overflow-y: auto)  │
│    ·참여자 목록      │                                         │
│    고정 너비         ├─────────────────────────────────────────┤
│    overflow-y: auto │          입력 영역 (min 72px)            │
└─────────────────────┴─────────────────────────────────────────┘
```

### 2.2 GNB (AppGnb.vue)

```
┌────────────────────────────────────────────────────────────────┐
│ 🗨 ChatterAI        [검색창 (비활성, 클릭 시 확장)]  [🔔] [👤]  │
└────────────────────────────────────────────────────────────────┘
```

| 요소 | 설명 |
|------|------|
| 로고 | `🗨 ChatterAI` 텍스트, 클릭 시 `/channels` 이동 |
| 검색창 | 클릭 시 채널·메시지 검색 모달 오픈 (MVP 외) |
| 알림 아이콘 | 🔔 클릭 시 알림 드롭다운 (MVP 외, 아이콘만 노출) |
| 프로필 아이콘 | 👤 클릭 시 드롭다운: 닉네임·역할 표시 + 로그아웃 버튼 |

**높이**: 64px 고정  
**배경**: `--bg-sidebar` (다크 배경, 로고 및 아이콘 흰색)

---

### 2.3 사이드바 (AppSidebar.vue)

```
┌──────────────────────────┐
│  채널               [+]  │  ← 관리자만 + 버튼 노출
│  ─────────────────────── │
│  # 일반             (3)  │  ← 안읽은 메시지 수 뱃지
│  # 공지                  │
│  # 자유                  │
│                          │
│  다이렉트 메시지    [+]  │
│  ─────────────────────── │
│  ● 김철수           (1)  │  ← 온라인 상태 점 + 뱃지
│  ○ 김영희                │
│                          │
│  ─────────────────────── │
│  [아바타] 홍길동  ● [⚙]  │  ← 하단 내 정보
└──────────────────────────┘
```

**채널 아이템 (ChannelItem.vue)**

| 상태 | 스타일 |
|------|--------|
| 기본 | 텍스트 `--text-on-dark`, `#` 접두사 |
| 호버 | 배경 `--bg-sidebar-hover` |
| 활성 (현재 채널) | 배경 `--bg-sidebar-active`, 텍스트 흰색 |
| 안읽은 메시지 있음 | 채널명 굵게(bold), 오른쪽 뱃지(파란 원형) |

**하단 내 정보 (UserProfile.vue)**

| 요소 | 설명 |
|------|------|
| 아바타 | 32px 원형, 이미지 없으면 닉네임 첫 글자 |
| 닉네임 | 13px, 최대 12자 말줄임 |
| 온라인 상태 점 | 아바타 우하단 오버레이 8px 원형 |
| 설정 아이콘 | ⚙ 호버 시 노출, 클릭 시 드롭다운 (로그아웃) |

## 3. 로그인 / 회원가입 페이지

### 3.1 로그인 페이지 (/login)

```
┌─────────────────────────────────────┐
│                                     │
│         🗨 ChatterAI               │
│                                     │
│   ┌─────────────────────────────┐  │
│   │  아이디                      │  │
│   └─────────────────────────────┘  │
│   ┌─────────────────────────────┐  │
│   │  비밀번호                    │  │
│   └─────────────────────────────┘  │
│                                     │
│   [에러 메시지 영역 (조건부 표시)]   │
│                                     │
│   ┌─────────────────────────────┐  │
│   │          로그인              │  │  ← primary 버튼
│   └─────────────────────────────┘  │
│                                     │
│   계정이 없으신가요? 회원가입        │
│                                     │
└─────────────────────────────────────┘
```

| 요소 | 스펙 |
|------|------|
| 카드 너비 | 400px, 화면 중앙 정렬 |
| 카드 배경 | `--bg-primary`, 그림자 `box-shadow: 0 4px 16px rgba(0,0,0,0.1)` |
| 인풋 | 높이 44px, border-radius 4px |
| 로그인 버튼 | 너비 100%, 높이 44px, `--color-primary` 배경 |
| 에러 메시지 | 빨간 텍스트 13px, 인풋과 버튼 사이에 노출 |

---

### 3.2 회원가입 페이지 (/join)

로그인 페이지와 동일한 카드 레이아웃.

| 필드 | 유효성 표시 방식 |
|------|-----------------|
| 아이디 | 4~20자 미만: 인풋 하단 빨간 텍스트 |
| 비밀번호 | 8자 미만 또는 조건 불충족: 인풋 하단 빨간 텍스트 |
| 비밀번호 확인 | 불일치: 인풋 하단 "비밀번호가 일치하지 않습니다." |
| 닉네임 | 2~20자 미만: 인풋 하단 빨간 텍스트 |

- 모든 필드 유효 전까지 **가입 버튼 비활성화** (opacity 0.5, cursor not-allowed)
- 가입 버튼 클릭 후 서버 에러(중복 등) → 해당 필드 하단에 에러 표시

## 4. 채널 채팅 페이지

### 4.1 채널 헤더 (ChannelHeader.vue)

```
┌────────────────────────────────────────────────────────────────┐
│  # 일반    참여자 12명          [📌 공지 내용 미리보기...]  [🔍] │
└────────────────────────────────────────────────────────────────┘
```

| 요소 | 스펙 |
|------|------|
| 채널명 | `# 채널명` 형식, 18px bold |
| 참여자 수 | 채널명 우측, 13px secondary 텍스트 |
| 공지 배너 | 공지 설정 시 헤더 우측에 `📌 공지내용(최대 50자 말줄임)` 표시, 클릭 시 해당 메시지로 스크롤 |
| 검색 아이콘 | 🔍 (MVP 외, 아이콘만 노출) |
| 높이 | 56px 고정 |
| 하단 구분선 | `1px solid --border-light` |

### 4.2 메시지 영역 (MessageList.vue)

#### 레이아웃

```
┌──────────────────────────────────────────┐
│  ─────────── 2025년 1월 15일 ──────────  │  ← 날짜 구분선
│                                          │
│  [아바타]  홍길동  14:30                  │  ← 첫 번째 메시지 (아바타/닉네임 표시)
│            안녕하세요!                   │
│            👍 3  😂 1                    │  ← 이모지 반응 바
│                                          │
│            반갑습니다!                   │  ← 연속 메시지 (아바타/닉네임 생략)
│                                          │
│  [🤖]  ChatterAI  14:30                 │  ← AI 응답 메시지
│  ┌─────────────────────────────────┐    │
│  │  안녕하세요! 무엇을 도와드릴까요? │    │  ← 연한 파란 배경
│  └─────────────────────────────────┘    │
│                                          │
│  ─── 홍길동님이 입력 중...               │  ← 타이핑 인디케이터
└──────────────────────────────────────────┘
```

#### 날짜 구분선

- 날짜가 바뀌는 첫 번째 메시지 위에 렌더링
- 텍스트: `YYYY년 MM월 DD일`, 양쪽에 가로선
- 오늘 날짜이면 `오늘`, 어제이면 `어제`로 표시

#### 메시지 묶음 규칙

```
isFirstInGroup = true  →  아바타(36px) + 닉네임(bold 15px) + 시각(13px secondary) 표시
isFirstInGroup = false →  아바타 영역 빈칸(36px), 닉네임·시각 생략
                          호버 시 시각만 좌측에 작게(11px) 표시
```

같은 사용자가 **5분 이내** 연속 전송 시 `isFirstInGroup = false`

#### 메시지 호버 액션 (MessageItem.vue)

메시지 영역에 마우스 오버 시 메시지 우측 상단에 액션 버튼 띄움.

```
[😊] [↩] [⋯]
```

| 버튼 | 기능 | 표시 조건 |
|------|------|-----------|
| 😊 이모지 | 이모지 피커 팝업 | 로그인 사용자 전체 |
| ↩ 답장 | (MVP 외, 버튼만 노출) | 로그인 사용자 전체 |
| ⋯ 더보기 | 드롭다운 메뉴 | 로그인 사용자 전체 |

**더보기 드롭다운 메뉴 구성**

| 항목 | 표시 조건 |
|------|-----------|
| 메시지 수정 | 본인 메시지 + 삭제되지 않음 + AI 메시지 아님 |
| 메시지 삭제 | 본인 메시지 또는 관리자 + 삭제되지 않음 |
| 공지로 지정 | 관리자 전용 |

#### AI 메시지 스타일

```css
.message-ai {
  background: var(--color-ai);       /* #F0F4FF */
  border: 1px solid var(--color-ai-border); /* #C7D8F8 */
  border-radius: var(--border-radius-md);
  padding: 10px 14px;
  margin-left: 44px;  /* 아바타 너비만큼 들여쓰기 */
}
```

아바타: `🤖` 이모지 36px 원형 배경, 닉네임: `ChatterAI` bold

#### AI 로딩 메시지

```
[🤖]  ChatterAI
      ● ● ●  (점 3개 애니메이션, 0.4초 간격으로 순차 페이드)
```

#### 삭제된 메시지

```
🚫 삭제된 메시지입니다.
```

이탤릭체, `--text-secondary` 색상. 이모지 반응·첨부파일 영역 숨김.

#### 새 메시지 뱃지 (NewMessageBadge.vue)

사용자가 스크롤을 위로 올린 상태에서 새 메시지 수신 시:

```
┌─────────────────────────┐
│  ↓  새 메시지 3개       │  ← 메시지 영역 하단 중앙에 고정
└─────────────────────────┘
```

클릭 시 최하단으로 스크롤 + 뱃지 사라짐

### 4.3 메시지 입력 영역 (MessageInput.vue)

```
┌────────────────────────────────────────────────────────────────┐
│  [📎]  메시지를 입력하세요... (#일반)               [😊] [▶]  │
│                                                                │
│  ┌───────────────────────────────────┐                        │
│  │ 📎 이미지.jpg (204KB)  ×          │  ← 첨부 미리보기 영역  │
│  └───────────────────────────────────┘   (파일 첨부 시 노출)  │
└────────────────────────────────────────────────────────────────┘
```

| 요소 | 스펙 |
|------|------|
| 외곽 테두리 | `1px solid --border-medium`, border-radius 8px, margin 16px |
| 텍스트 입력창 | `contenteditable` 또는 `<textarea>`, 최소 1줄~최대 10줄 자동 확장 |
| 플레이스홀더 | `메시지를 입력하세요... (#채널명)` |
| 📎 파일 첨부 | 좌측 하단, 클릭 시 파일 선택 dialog |
| 😊 이모지 | 우측 하단, 클릭 시 이모지 피커 팝업 |
| ▶ 전송 버튼 | 우측 하단, 내용 없으면 비활성(회색) |
| @AI 하이라이트 | 입력 중 `@AI` 문자열을 파란색 bold로 실시간 강조 |

**단축키**
- `Enter`: 전송
- `Shift + Enter`: 줄바꿈
- `ESC`: 인라인 수정 모드 취소

**파일 첨부 미리보기 (FilePreview.vue)**

```
┌────────────────────────────────────────────┐
│  🖼️ [썸네일]  이미지파일.jpg (204KB)  [×]  │  ← 이미지
│  📄            문서파일.pdf  (1.2MB)  [×]  │  ← 일반 파일
└────────────────────────────────────────────┘
```

- `[×]` 클릭 시 해당 파일만 미리보기에서 제거
- 최대 5개 초과 시 파일 선택 disabled

### 4.4 타이핑 인디케이터 (TypingIndicator.vue)

메시지 입력 영역 상단 (입력창 바로 위)에 표시.

```
홍길동님이 입력 중...
```

- 폰트: 13px italic, `--text-secondary` 색상
- 2명: `홍길동, 김철수님이 입력 중...`
- 3명 이상: `여러 사람이 입력 중...`
- 3초간 새 타이핑 이벤트 미수신 시 자동 사라짐

### 4.5 이모지 피커 (EmojiPicker.vue)

메시지 호버 액션 또는 입력창 이모지 버튼 클릭 시 팝업.

```
┌────────────────────────────────┐
│  자주 쓰는 이모지               │
│  👍 ❤️ 😂 🎉 🙏 😮 😢 🔥      │
│                                │
│  전체                          │
│  😀 😃 😄 😁 😆 😅 🤣 😂     │
│  ...                           │
└────────────────────────────────┘
```

- 팝업 위치: 트리거 버튼 위쪽 (viewport 상단이면 아래쪽)
- 팝업 외부 클릭 시 닫힘
- 자주 쓰는 이모지: 로컬스토리지에 최근 사용 8개 저장

### 4.6 이모지 반응 바 (ReactionBar.vue)

메시지 하단에 반응이 있을 때만 렌더링.

```
[👍 3] [😂 1] [+]
```

| 상태 | 스타일 |
|------|--------|
| 기본 | 배경 `--bg-secondary`, 테두리 `--border-light`, border-radius full |
| 내가 반응한 이모지 | 배경 `#E8F0FF`, 테두리 `--color-primary` |
| 호버 | 배경 살짝 어둡게 |

- `[+]` 버튼 클릭 시 이모지 피커 팝업 (EmojiPicker 재사용)
- 반응 아이템 클릭 시 토글 (추가 또는 취소)
- 반응 아이템 호버 시 "OOO님 외 N명이 반응했습니다" 툴팁

### 4.7 메시지 인라인 수정 (MessageEditInline.vue)

더보기 메뉴 > 수정 클릭 시 해당 메시지가 인라인 편집 모드로 전환.

```
┌──────────────────────────────────────────────┐
│  수정된 내용을 입력하세요                      │  ← textarea, 기존 내용 채워짐
└──────────────────────────────────────────────┘
ESC로 취소  •  Enter로 저장
```

- 기존 메시지 내용이 `textarea`에 자동 입력
- `Enter`: 저장 (내용 변경 없으면 비활성)
- `ESC`: 취소 (원래 메시지 복원)
- 하단 안내 텍스트: `ESC로 취소 · Enter로 저장` (11px secondary)

## 8. 버튼 표시 조건 매트릭스

### 8.1 메시지 더보기 메뉴

| 항목 | 비회원 | 일반 회원 (본인 메시지) | 일반 회원 (타인 메시지) | 관리자 |
|------|--------|------------------------|------------------------|--------|
| 수정 | ❌ | ✅ (삭제·AI 아님) | ❌ | ✅ (AI 아님) |
| 삭제 | ❌ | ✅ | ❌ | ✅ |
| 공지로 지정 | ❌ | ❌ | ❌ | ✅ |
| 이모지 반응 😊 | ❌ | ✅ | ✅ | ✅ |

### 8.2 사이드바 버튼

| 버튼 | 비회원 | 일반 회원 | 관리자 |
|------|--------|----------|--------|
| 채널 `+` 버튼 | ❌ | ❌ | ✅ |
| DM `+` 버튼 | ❌ | ✅ | ✅ |

### 8.3 전송 버튼

| 조건 | 버튼 상태 |
|------|-----------|
| 입력창 비어있음 | 비활성 (opacity 0.5, 클릭 불가) |
| 텍스트 입력됨 | 활성 |
| 파일만 첨부됨 | 활성 |

### 8.4 회원가입 가입 버튼

| 조건 | 버튼 상태 |
|------|-----------|
| 필드 하나라도 비어있음 | 비활성 |
| 유효성 검사 통과 안 됨 | 비활성 |
| 모든 필드 유효 | 활성 |

### 8.5 관리자 회원 관리 버튼
